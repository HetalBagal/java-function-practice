# Java Function Practice

This repository contains basic Java programs to understand and practice functions (methods).

## 📌 Programs Included

1. Print Name using Function
2. Sum of Two Numbers
3. Product of Two Numbers
4. Factorial of a Number

## 🧠 Concepts Covered

- Functions (Methods)
- User Input using Scanner
- Loops
- Conditional Statements

## 💻 Technologies Used

- Java
- VS Code / IntelliJ IDEA

## ▶️ How to Run

1. Compile the program:
   javac FileName.java

2. Run the program:
   java FileName
